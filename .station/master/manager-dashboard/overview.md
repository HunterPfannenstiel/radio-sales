# Overview

At the top level, the manager wants a list that helps them triage. The best view is not “all the metrics,” but “who is healthy, who is drifting, and who is failing for a specific reason”

Use the dashboard as a prioritization tool, not a complete record. A rep should bubble up if they are behind pace, projected to miss goal, or have a sharp mismatch between activity and results, such as high activity but weak conversion.

A good rule is to highlight only the few dimensions that explain most performance variance: volume, conversion, and deal size. Everything else should not be on the main list.
