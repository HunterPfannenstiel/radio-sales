# Overview

Core Question: "Why is this person behind?"

The best diagnostic set is small and causal: activity pace, asks, closing ratio, average account value

That structure supports the manager’s real job, which is identifying whether the gap is caused by low volume, poor conversion, smaller deal size, or weak follow-through

Also avoid showing only activity counts without outcome context. Managers care less about “how busy” a rep was than whether the activity is producing meetings, opportunities, and sold revenue.
