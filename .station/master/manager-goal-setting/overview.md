# Overview

A manager sets the accountability baseline for each rep: a monthly sales goal and weekly activity targets (calls per week, asks per week). These numbers flow into the rep's dashboard and drive all pace and percent-to-goal calculations.

The manager owns all values. When any value changes, the system immediately recalculates against the new number — current truth only, no historical snapshots.

## Core Information

Monthly sales goal.

Calls per week target.

Asks per week target.

## Rules

Goals and targets can be changed at any time, including mid-month or mid-week. Changes take effect immediately for the current period.

Each rep belongs to one manager at a time for goal-setting purposes.

No notifications are sent when values change.

## Objective

A demoable feature, not a functional one.
